# RSA 암호화 알고리즘 C언어 구현 
RSA암호화 기법의 수학적 원리를 이해할 수 있게되었고 실제로 암호화 복호화가 작동하는 프로그램을 제작할 수 있었다.
키를 만들거나 암호화와 복호화하는 과정에서 크기가 큰 정수를 사용해야하지만 메모리의 한계가 있어 크기가 작은 소수와 작은 숫자만을 암호화, 복호화 시킬 수 있었던 점이 아쉽다.
